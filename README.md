# Phoenix Innovative Technologies Website

A multi-page website for Phoenix Innovative Technologies Ltd, featuring service pages, about pages, contact form with email integration, and an AI chatbot.

## Features

- Multi-page website with services and about sections
- Responsive design with Bootstrap 5
- Custom animations and transitions
- Contact form with email integration (Nodemailer)
- AI chatbot for visitor assistance
- WhatsApp integration
- Social media links
- Sticky navigation header
- Floating action buttons (WhatsApp & Chatbot)

## Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** EJS, Bootstrap 5, CSS3
- **Email:** Nodemailer (Gmail)
- **Session Management:** express-session
- **Animations:** AOS (Animate On Scroll), Custom CSS
